CAS P&L Excel Load Help

This page allows you to load branch and consolidated P&L totals from an Excel spreadsheet. You can load anything that is already on file or you can browse and load a new file. To reload a file that is already in the folder, use the Review link to the left of the file name.

Using Excel and saving on the H: drive, you have to save the Excel file as Tab Delimted Text, close the Excel file, use the Browse button to find the file on the H: drive, press the OK button to review the changes, and finally press Update to load the P&L totals.

Find the saved Excel P&L file you want to load using the Browse button then press the OK button to load the data in the Excel file to this page.


The spreadsheets are created by running the NewVison system and using the General Ledger reports. The report PFC - Excel Branch Account Schedule will produce the spreadsheets that are to be loaded into this page. You produce 1 P&L spreadsheet for each branch and 1 consoldated.

The basic process is to:
Find the file you want to load.
Press the OK button to see the contents on this page.
If it is all OK, use the Update button to actually update the system.

P&L data already on file for a month/branch will be replaced.

Canada Bill Of Lading Help


This page is used to generate Canada Bill Of Lading Summary and/or Detail Reports.

- Simply enter a Bill of Lading Number and click ViewReport

- The Summary Report will be displayed for the specific B/L Number entered. The data can be exported to an Excel file format by clicking Export To Excel

- Click List to display the Detail Report for the specific B/L Number. The Detail data can also be exported to an Excel file format by clicking Export To Excel

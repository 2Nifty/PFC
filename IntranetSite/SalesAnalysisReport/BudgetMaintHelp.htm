CSAR Budget Import Help

This page allows you to do mass loads for YTD Customer Budgets from an Excel spreadsheet. The spreadsheets have to be put in a specific folder to be loaded. The folder is on the page behind this one. You can load anything that is already in this folder or you can browse and load a new file. To reload a file that is already in the Contract folder, use the Review link to the left of the file name.

If you are using Excel and making changes to a file on the H: drive, you have to copy, review, and update the YTD Budgets in the system. You have to save the Excel file, close the Excel file, use the Browse button to find the file on the H: drive, press the OK button to review the changes, and finally press Update to update the budgets.

Enter the Month and Year, find the Excel Customer YTD Budget file you want to load using the Browse button then press the OK button to load the budgets in the Excel file to this page. If you do not see the file you just copied, refresh the page using the refresh button for the browser.

The basic process is to:
Enter the Month and Year for the Budget file you are loading.
Find the file you want to load.
Press the OK button to see the spreadsheet contents on this page.
If it is all OK, use the Update button to actually update the system.

Budgets already on file for a customer will be replaced.

You have to use a special spreadsheet that has the right layout and names. Using the wrong spreadsheet will cause the process to fail. The sample below has the proper column names. The Customer column has to be formatted as Text.
